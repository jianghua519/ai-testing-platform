# AI自动化测试工具主要技术选型测试计划

## 测试范围

验证主要技术选型文档是否完整覆盖：

- 控制面
- Web Console
- Agent
- Web 执行
- API 执行
- AI 编排
- 记忆系统
- 数据与基础设施
- 可观测性
- 插件扩展

## 覆盖风险

- 技术选型只给出名词，没有说明理由。
- 技术选型与上一轮架构设计脱节。
- 文档链路不完整。
- 校验脚本执行失败。

## 测试项

1. 检查项目任务说明、设计说明、测试计划、测试报告、测试举证是否齐全。
2. 检查技术选型文档是否为主要模块给出“推荐 + 理由 + 当前不选”。
3. 检查技术选型是否与上一轮架构设计一致。
4. 运行 `bash ./scripts/validate_docs.sh`。
5. 运行 `bash ./scripts/validate_contracts.sh`。

## 通过标准

- 主要技术模块均有明确选型。
- 技术选型文档能够支撑实现阶段决策。
- 文档和契约校验通过。
- 测试报告与证据记录可追溯。
