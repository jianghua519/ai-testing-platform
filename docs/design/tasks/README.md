# 任务设计说明索引

每个正式任务都应在这里生成一份设计说明，记录背景、方案、风险和验证计划。

推荐命名：

- `YYYY-MM-DD-<scope>-design-task.md`

可以通过 `bash ./scripts/create_delivery_bundle.sh "请做xxx"` 自动生成。
